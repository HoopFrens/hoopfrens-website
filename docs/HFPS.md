# Hoop Frens Platform System

HFPS is the internal foundation for the Hoop Frens Executive Workspace.

The approved architecture centers the workspace around five core business objects:

1. Knowledge
2. Projects
3. Assets
4. People & Organizations
5. Decisions

This foundation pass creates the folders, TypeScript contracts, service boundaries, and documentation placeholders needed before production behavior is added.

No public-facing pages, production AI calls, or deployment changes are introduced in this pass.
