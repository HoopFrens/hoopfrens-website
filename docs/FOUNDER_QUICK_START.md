# Founder Quick Start

1. Open the repo:

```bash
cd /Users/antwonewilliams/Documents/GitHub/hoopfrens-website
```

2. Start the local app:

```bash
npm run dev
```

3. Log in:

```text
http://localhost:3000/admin/login
```

Use an approved Firebase admin account.

4. Open Headquarters:

```text
http://localhost:3000/executive-workspace
```

5. Create a project:

```text
Spotlight Ashland University
```

6. Continue the project:

```text
Continue Ashland
```

7. Review:

Select `Review Project`, then review the package.

8. Approve:

Select `Approve`, then `Confirm Approval`.

9. Return:

Use `Return to Headquarters` to return to the Executive Office.

This local preview is session-only. No AI, Firestore persistence, publishing, or production workflow runs.
